\section{Reading perf files}
In this section, a description is given of how the perf data file can be read. For this, an application named readperf\index{readperf} is presented. The goal of readperf is not to be used as a tool to analyze the data file, as perf report can be used for this. It is meant to show how the data file can be processed. In addition, it is proof that the data format is understood.
